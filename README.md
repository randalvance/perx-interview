# Perx React App

[![Build Status](https://randalvance.visualstudio.com/perx-interview/_apis/build/status/randalvance.perx-interview?branchName=master)](https://randalvance.visualstudio.com/perx-interview/_build/latest?definitionId=10&branchName=master)

This project was built for the [Perx React Interview](https://github.com/PerxTech/react-interview).

The app was deployed to [https://perx-react-randal.azurewebsites.net/](https://perx-react-randal.azurewebsites.net/).

# What was implemented in this project?

1. React with Typescript
2. Redux and Sagas
3. Hooks
4. Github Integration
5. Angular Material for React
6. Unit Tests with Jest
7. Linting with TSLint and Prettier
8. CI/CD Pipeline
9. Deployment of app to a Docker container in Azure
10. Responsive Design

# How to Run the project

1. `npm install`
2. `npm start`
3. App is accessible at `http://localhost:5000`
